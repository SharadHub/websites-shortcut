# Websites Shortcut

A clean, modern, and high-speed web hub designed to provide one-click access to your most-used daily platforms. This project eliminates the need for searching or managing messy bookmarks by consolidating social media, freelance work, video streaming, and music services into a single, responsive interface.

## üöÄ Features

* **Centralized Dashboard**: Quick links to major daily platforms like Facebook, YouTube, Upwork, Netflix, and Spotify.
* **Modern UI/UX**: A sleek "Dark Mode" aesthetic using a deep slate palette (`#0f172a`) with vibrant blue accent colors.
* **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices using CSS Flexbox.
* **Interactive Elements**: 
    * **Glassmorphism Header**: A frosted-glass navigation bar with a blur effect that stays fixed as you scroll.
    * **Dynamic Hover Effects**: Social and music buttons respond to user interaction with scaling and brand-specific color transitions.
    * **Work Platform Cards**: A vertical flex row layout for freelance platforms including high-visibility text and direct-action buttons.

## üìñ How to Use

1. **Clone or download** the project files to your local machine.
2. **Ensure your image assets** (GIFs and JPEGs) are in the same directory as the HTML file.
3. **Open `webShortcut.html`** in any modern web browser.
4. **Click any category** in the navigation bar to jump directly to that section via smooth-scroll anchors.
